Generate Windows Reports

Run these commands on your Windows PC to generate diagnostic reports, then drag the files back here for analysis.

âš¡ Power & Battery

Battery Report

Admin

Battery health, design vs full charge capacity, cycle count, recent usage history.

powercfg /batteryreport /output "%USERPROFILE%\Desktop\battery-report.html"
Format: HTML

Energy Report

Admin

Power policy errors, warnings, informational items affecting energy efficiency.

powercfg /energy /output "%USERPROFILE%\Desktop\energy-report.html"
Format: HTML

Sleep Study

Admin

Sleep sessions, battery drain rate during sleep, top offenders, wake sources.

powercfg /sleepstudy /output "%USERPROFILE%\Desktop\sleep-study.html"
Format: HTML

Power Schemes

Active power plan and configured power settings.

powercfg /query > "%USERPROFILE%\Desktop\power-schemes.txt"
Format: TXT

ğŸ’» System & Hardware

MSInfo32

Complete system information: OS, CPU, RAM, GPU, drivers, devices, conflicts.

msinfo32 /report "%USERPROFILE%\Desktop\msinfo32.txt"
Format: TXT

DxDiag

GPU details, DirectX version, display driver, audio devices, problems found.

dxdiag /t "%USERPROFILE%\Desktop\dxdiag.txt"
Format: TXT

System Info

OS version, boot time, total RAM, network config, installed hotfixes.

systeminfo > "%USERPROFILE%\Desktop\systeminfo.txt"
Format: TXT

Driver Query

All installed drivers with version numbers, dates, and status.

driverquery /v /fo csv > "%USERPROFILE%\Desktop\drivers.csv"
Format: CSV

ğŸ’¾ Storage

Disk Info

Disk model, size, and health status for all drives.

wmic diskdrive get model,size,status /format:csv > "%USERPROFILE%\Desktop\disk-info.csv"
Format: CSV

Volume Info

Partition sizes and free space for all volumes.

wmic volume get caption,capacity,freespace /format:csv > "%USERPROFILE%\Desktop\volume-info.csv"
Format: CSV

ğŸŒ Network

WiFi Report

Admin

WiFi connection history, disconnects, signal quality, error events.

netsh wlan show wlanreport
Format: HTML (saved to ProgramData)

Network Config

IP addresses, DNS servers, DHCP, all network adapters.

ipconfig /all > "%USERPROFILE%\Desktop\network-config.txt"
Format: TXT

Firewall Status

Firewall state for all network profiles (Domain, Private, Public).

netsh advfirewall show allprofiles > "%USERPROFILE%\Desktop\firewall-status.txt"
Format: TXT

ğŸ›¡ï¸ Security

Installed Updates

All installed Windows hotfixes, KB numbers, and install dates.

wmic qfe list full /format:csv > "%USERPROFILE%\Desktop\updates.csv"
Format: CSV

System Events

Admin

Recent system event log entries: errors, warnings, BSODs.

wevtutil qe System /c:100 /f:xml /rd:true > "%USERPROFILE%\Desktop\system-events.xml"
Format: XML

SFC Scan

Admin

System File Checker integrity verification results.

sfc /verifyonly > "%USERPROFILE%\Desktop\sfc-scan.txt"
Format: TXT

ğŸš€ Performance

Perfmon Report

Admin

Comprehensive CPU, disk, memory, and network diagnostics.

perfmon /report
Format: HTML

Startup Programs

All programs that run at boot with their command paths.

wmic startup get caption,command /format:csv > "%USERPROFILE%\Desktop\startup.csv"
Format: CSV

Running Processes

Active processes with memory usage and CPU time.

tasklist /v /fo csv > "%USERPROFILE%\Desktop\processes.csv"
Format: CSV
